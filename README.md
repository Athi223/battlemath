# BattleMath

A simple browser based game that helps children learn the four basic fundamentals of math

- Subtraction
- Addition
- Multiplication
- Division

### Built With

- [React](https://reactjs.org/)
- [Cypress](https://www.cypress.io/)

### Quick Start

```sh
git clone git@github.com:JesseRWeigel/battlemath.git
```

```sh
cd battlemath

yarn install

yarn start
```

### Publish

```sh
npm run build
```
